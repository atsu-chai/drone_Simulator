# ドローン国家資格 実地試験対策シミュレーター

仕様書 `drone_spec_v1.txt` をもとにしたブラウザ版プロトタイプです。

## 起動

`index.html` をブラウザで開きます。

## 実装済み

- 横画面前提の飛行訓練画面
- 飛行前点検の順序チェック
- 練習モード / 本番モードの切り替え
- Gamepad API による外部コントローラー検出
- Switch / PlayStation / Xbox / 汎用向けプリセット選択
- 離陸、移動、8の字、着陸の一連フロー
- 高度ブレ、着陸時の流れ、点検ミス、反応時間を使った簡易採点
- 合否、スコア、所要時間、改善点の表示

## 開発者モード

外部コントローラーがない環境で確認するため、開発者モードではキーボードで操作できます。初期状態では ON です。

- `W` / `S`: 前後移動
- `A` / `D`: 左右移動
- `E` / `Q`: 上昇 / 下降
- `←` / `→`: 旋回

製品仕様では、飛行訓練は外部コントローラー接続を前提とします。
